## 3. 一个小组件（MDX 里直接写）

export function Callout(props) {
  return (
    <div className="my-6 rounded-xl bg-amber-50 p-4 text-amber-900 ring-1 ring-amber-200">
      <div className="text-sm font-semibold">提示</div>
      <div className="mt-1 text-sm leading-6">{props.children}</div>
    </div>
  );
}

<Callout>
  后续我会把 **Giscus 评论组件**插到文章页底部（依赖仓库开启 Discussions）。
</Callout>

## 4. 你接下来写文章的方式

- Markdown：放在 `src/content/blog/*.md`
- MDX：放在 `src/content/blog/*.mdx`

只要 frontmatter（title/description/pubDate 等）符合 content schema，就会自动出现在列表里。
